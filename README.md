# Fresh Food Market

> Eat healthy, live well!

## Installation
```shell
$ git clone https://github.com/MarwanYoussef/ACML_Final
$ cd ACML_Final
```

> now install npm packages for server and connect to database
```shell
$ cd Server
$ npm install
$ node app.js
```

> now in another command line install npm packages for client
```shell
$ cd Client
$ npm install
$ ng serve -o
```
